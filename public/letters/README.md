# Assets lettres officielles FRMT

Images extraites du modèle Word « Lettre de réservation Centre National » :

| Fichier | Usage |
|---------|--------|
| `frmt-header-logo.png` | Logo en-tête (tableau Word) |
| `frmt-cachet.jpeg` | Cachet / chachet près de la signature |
| `image1.png`, `image2.jpeg` | Copies sources (référence) |

La génération PDF/DOCX charge ces fichiers via `lib/letters/load-letter-assets.server.ts`.
