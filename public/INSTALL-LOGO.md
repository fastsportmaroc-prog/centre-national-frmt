# Logo officiel FRMT

## Méthode rapide

1. Enregistrez l’image du logo officiel (fichier reçu par email ou fourni par la FRMT) sous le nom :
   **`frmt-logo.png`**
2. Placez-le dans ce dossier : `tennis-center/public/frmt-logo.png`
3. Rechargez l’application (**Ctrl + F5**)

Le logo apparaîtra sur les **rapports**, **impressions**, **PDF** et dans la **barre latérale**.

## Ligne de commande (optionnel)

```bash
cd tennis-center
node scripts/install-frmt-logo.mjs "C:\chemin\vers\votre-logo-officiel.png"
```
