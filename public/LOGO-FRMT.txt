Logo officiel FRMT pour les rapports
=====================================

Source unique :

  public/logo-frmt.png

Régénérer le base64 embarqué (après remplacement du PNG) :

  node scripts/convert-logo-base64.js

Ce logo est utilisé pour :
- Impression des rapports (window.print)
- Export PDF jsPDF
- Documents budget prévisionnel
- API /api/frmt-logo

Le fichier public/frmt-logo.png est un alias du même PNG (compatibilité).
